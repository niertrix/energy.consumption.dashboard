# Energy Consumption Dashboard
An interactive dashboard project created on Microsoft Excel

# Dataset Used
<a href=https://github.com/niertrix/energy.consumption.dashboard/blob/main/Energy%20Consumptions%20Dataset.xlsx>Energy Consumptions Dataset</a>

# KPIs
- What are the total costs and consumption?
- What are the average costs and consumption?
- How much cost and consumption does each type of energy source constitute?
- What is the distribution of average energy costs and consumption across different cities/states?
- How does the rate of costs and consumption change across each year?
- Which buildings have the highest amount of costs and consumption?
- What is the total cost and consumption per month?

# Process
1. Clean the dataset and check for any null or corrupt data.
2. Make sure all data types and values are consistent.
3. Create data model from cleaned data.
4. Calculate required measures for reporting.
5. Create pivot tables according to the asked questions.
6. Create visualizations for these pivot tables.
7. Cleanly gather the visuals into a single dashboard.
8. Duplicate the first dashboard and have one dashboard for cost measures and the other for consumption measures.

# Insights
- Chicago has the highest rate of energy consumption.
- Water energy sources have the highest cost and consumption.
- Energy consumption is generally always increasing except in 2017 when consumption was actually decreasing.
- Energy costs have been fairly stable across all recorded years.
